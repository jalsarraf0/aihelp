# .CLAUDE.md — aihelp
# Last updated: 2026-03-11 | Evidence: Cargo.toml, src/*.rs, tests/

## What It Is
Rust CLI (v0.3.1) for querying LM Studio models from the terminal.
Single binary. Supports stdin piping, streaming responses, and MCP tool discovery.
Default endpoint: `http://192.168.50.2:1234`
Config: `~/.config/aihelp/config.toml`

## Source Files (src/)
| File | Role |
|---|---|
| `main.rs` | `main()` — calls `aihelp::run(cli)` via Clap parse |
| `lib.rs` | Library re-exports, `Cli` struct, `run()` dispatch |
| `client.rs` | `OpenAiClient` — LM Studio HTTP client (streaming + non-streaming) |
| `agent.rs` | `run_agent()` — MCP multi-turn agentic loop |
| `mcp.rs` | `McpBackend` trait, tool discovery, tool invocation |
| `config.rs` | Config load/save to `~/.config/aihelp/config.toml` |
| `prompt.rs` | `build_user_message()`, `StdinContext`, `SYSTEM_PROMPT` |
| `setup.rs` | Interactive setup wizard |

## Build / Install / Run
```bash
cargo build --release                  # build
cargo install --path .                 # install to ~/.cargo/bin/aihelp
aihelp --setup                         # interactive config wizard
aihelp --list-models                   # list models from LM Studio
aihelp "your question"                 # basic query
cat file | aihelp "explain this"       # stdin piping
aihelp --mcp "use tools"               # MCP-enabled query
```

## MCP Policy (3 modes — set in config.toml)
- `read_only` (default) — read-only MCP operations only
- `allow_list` — explicit tool allowlist
- `all` — unrestricted; do NOT set as a code default

## Agentic Loop (agent.rs)
`run_agent()` takes `AgentRunOptions { model, stream, json, dry_run, quiet, mcp_enabled,
mcp_max_tool_calls, mcp_max_round_trips }`. Without MCP → single turn. With MCP →
multi-turn loop capped by `mcp_max_tool_calls` and `mcp_max_round_trips`.

## Hard Conventions
- `anyhow::Result` throughout — no `.unwrap()` in library code.
- Async: `tokio` multi-thread only — no `std::thread::spawn` for I/O.
- TLS: `rustls` only — do NOT add `openssl` features.
- Config: TOML only — do NOT introduce JSON or YAML config.
- New CLI flags → `clap` derive macros in `lib.rs`.
