# .GEMINI.md - aihelp

This is a Rust-based CLI tool for interacting with an LM Studio (or any OpenAI-compatible) endpoint from the terminal.

### Key Directories & Files
- `src/main.rs`: The main binary entrypoint.
- `src/lib.rs`: The core library logic.
- `src/client.rs`: Handles communication with the LLM endpoint.
- `src/mcp.rs`: Implements the Model Context Protocol for tool use.
- `src/config.rs`: Manages user configuration.
- `tests/`: Integration and regression tests.
- `Cargo.toml`: Project manifest and dependencies.

### Core Commands
- `cargo build --release`: Compile the optimized binary.
- `cargo run -- <args>`: Run the application from source (e.g., `cargo run -- --help`).
- `cargo install --path .`: Install the `aihelp` binary to `~/.cargo/bin/`.

### Validation Commands
- `cargo test`: Run the full suite of unit and integration tests.
- `cargo clippy -- -D warnings`: Run the clippy linter and treat all warnings as errors.
- `cargo fmt -- --check`: Check for formatting issues.

### Usage
- `aihelp --setup`: Run the first-time setup wizard to configure the endpoint and default model.
- `aihelp "your prompt"`: Send a simple prompt.
- `cat file.txt | aihelp "summarize this"`: Pipe content into the prompt.
- `aihelp --mcp "use tools to answer"`: Run a prompt with MCP tool-use enabled.
- `aihelp --list-models`: List available models from the configured endpoint.
